# TaskForge React

A modern task management dashboard built with React and Tailwind CSS.

## Features

* Dashboard overview
* Project management
* Task management
* Search and filters
* Responsive design
* Local storage support

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router DOM
* React Icons

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd taskforge-react
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.
